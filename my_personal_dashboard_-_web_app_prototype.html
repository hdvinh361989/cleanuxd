ABOUT THE PROJECT

Thanks to the development of technology, more and more apps have been built up and introduced to support users' life and work activities. However, from the fact that users have to use different services in different apps to accomplish their needs, it creates difficulties plus inconvenience to switch between those. As a result, the more apps they use, the more efforts that users have to put into to manage.  .

From the fact above, My Personal Dashboard (or TU), an all-in-one integrated platform, will combine all the user's personal digital activities to be into one place.

There are four main sections within TU (in version 1.1):

•  Section 1 - My Boards: allows the user to create boards to add integration from their original apps in order to manage from TU

•  Section 2 - My Timeline: allows the user to record their daily journal in more personal and customizable ways

•  Section 3 - Discovery: allows the user to discover and follow public boards of integration from TU community

•  Section 4 - Account Settings: allows the user to change their account information and notification settings
