ABOUT THE PROJECT

Tribal Blast is a shooting mobile game application that will allow multiple players to compete with each other.  The app has a beta version ready in September 2019.  The customer approached me for further help on the game testing.  So I organized a Game Testing Day with 13 players to review the app's compatibility and performance.

I have to say that we had soooooo much fun on this Game Testing Day. We had an active and exciting game playing in the city's zoo.

In the beginning, I expected that the players would go through most of the Test Cases successfully. However, there were a number of factors that happened including bugs, crashings, technical limitations, the game ideas, the UX gaps, etc. All the things made the Game Testing Day outcomes become so insightful for me and the app owners.


Currently, Tribal Blast has been removed from the Google App Store 😢.

But this is a very special project that I have worked on which I have learned and collected valuable UX lessons.

My general approaches  in this project were:

1. Player Recruitment

2. Compatibility Testing

3. Preliminary Interviews

4. Test Cases

5. Play Day

6. Player Interviews

7. Report Summary
