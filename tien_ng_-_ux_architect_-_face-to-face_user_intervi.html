This project is quite similar to the Tiki project, but it is more focused on Information Architecture.

Before the Interview Day, I have sent an online questionnaire form to the target user to collect further individual experience towards their using Mobile Banking app.

On the Interview Day, I prepared 05 UX Exercises with the cards to watch and observe how the users interact with the data.

 

The UX Exercises are about eliminating, sorting, voting for the types of information the user is actually using in the app. 

We had a great time talking to the users and gained so much valuable feedback for the app's information organization.

ABOUT THE PROJECT

This is a project to revamp a current mobile app for a top-tier bank in Vietnam. I took part in the role of Information Architect and worked closely with the Project Manager (the talented-cheerful-pink man in the photo). We hosted two-day interviews with 12 target customers working through several UX Exercises to understand the user behaviors and opinions.
