Face-to-face User Interviews on Tiki mobile app project

Understand the market, understand the competitors and understand the customers. Those are 3 important things before you build up the product. In fact, there are many types of User Experience Research Methods such as: User Surveys, User Interviews, Focus Groups, etc.

And all of this would depend on the project's scope and budget.

At the end of Stage 2, user's personas and insights should be found in order to support the next stage of development.


Information architecture (IA) is the system and structure that is used to organize and label content on the website, app or product.

In stage 2, it will show how users look for information and what type of information is critical. From that, I will go through multiple techiques

in order to create an effective IA.

At the end of stage 3, we will be ready to work on the prototype design.

All the study and analysis above will serve as a design base for the prototype. There will be multiple iterations of prototype deliverables according to the client's feedback. When the design work is completed, the prototype should have a Specification Document along in order to support the UI Designer and Developers understand the product's UX design.

For the project success, I always make sure that the final outcome will be completed with 100% client's satisfaction.
